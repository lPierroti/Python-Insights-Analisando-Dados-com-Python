# Estudo de análise de dados com Python

Projeto desenvolvido em Python com foco em análise de dados e identificação de padrões relacionados ao cancelamento de clientes (churn).

O objetivo do projeto é explorar os dados, identificar comportamentos relevantes e gerar insights que possam auxiliar na compreensão dos fatores associados à evasão de clientes.

---

## Tecnologias utilizadas

- Python
- Pandas
- Plotly
- Jupyter Notebook

---

## Principais atividades realizadas

- Tratamento e limpeza de dados
- Análise exploratória
- Identificação de padrões de cancelamento
- Visualização de dados
- Geração de insights a partir dos dados analisados

---

## Objetivo do projeto

Desenvolver conhecimentos práticos em análise de dados utilizando Python, aplicando técnicas de manipulação, visualização e interpretação de dados em um cenário próximo ao ambiente corporativo.

---

## Aprendizados

Durante o desenvolvimento deste projeto, foram praticados conceitos relacionados a:

- análise de dados;
- manipulação de datasets;
- visualização gráfica;
- identificação de padrões;
- resolução de problemas com Python.
